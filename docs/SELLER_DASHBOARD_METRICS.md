# Seller Dashboard Metrics

## Purpose

This document defines the metrics and reporting blocks that should appear in the advanced seller dashboard.

## Dashboard Goals

| Goal | Why It Matters |
|---|---|
| Show performance | Sellers need to know if the store is growing |
| Show money flow | Sellers need to understand revenue and payouts |
| Show product health | Sellers need to see what is selling and what is not |
| Show customer activity | Sellers need to understand buyer behavior |
| Show risk signals | Sellers need to act on issues early |

## Core Metrics

| Metric | Meaning |
|---|---|
| Gross sales | Total sales before deductions |
| Net sales | Sales after refunds and adjustments |
| Order count | Number of orders placed |
| Conversion rate | Percentage of visits or chats that become orders |
| Average order value | Average money spent per order |
| Refund rate | Percentage of orders refunded |
| Cancel rate | Percentage of orders canceled |
| Payout balance | Amount ready or pending payout |

## Product Metrics

| Metric | Meaning |
|---|---|
| Views | Product page or listing views |
| Add to cart | Number of times a product is added to cart |
| Orders per SKU | Orders generated by a product |
| Stock status | In stock, low stock, or out of stock |
| Approval status | Pending, approved, rejected |

## Customer Metrics

| Metric | Meaning |
|---|---|
| New customers | First-time buyers |
| Repeat customers | Buyers who purchase again |
| Customer location | Geographic distribution |
| Customer retention | How often customers return |
| Support contact rate | How often customers request help |

## Payments and Payout Metrics

| Metric | Meaning |
|---|---|
| Gross amount | Total before deductions |
| Commission | Platform fee deducted |
| VAT | Tax applied to eligible fees |
| Shipping contribution | Delivery-related deduction or support fee |
| Net payout | Amount owed to seller |
| Paid amount | Amount already transferred |
| Held amount | Amount currently blocked or in settlement |

## Risk Metrics

| Metric | Meaning |
|---|---|
| Return rate | Share of orders returned |
| Dispute rate | Share of orders under dispute |
| Delivery failure rate | Share of failed deliveries |
| Low stock alerts | Products that may go out of stock soon |
| Policy violations | Listings or actions that break rules |

## Suggested Layout

| Section | Content |
|---|---|
| Header summary | Revenue, orders, payout balance |
| Trend charts | Sales and order trends over time |
| Product table | Top products and stock status |
| Customer table | Repeat customers and geography |
| Finance section | Commission, VAT, payouts, holds |
| Risk panel | Returns, disputes, low stock, failures |

## Summary

The seller dashboard should help sellers make decisions, not just view numbers.
